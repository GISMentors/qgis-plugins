Creating UI¶

For now, we need two user inputs:

  • Input vector layer

  • Output target directory

Task

In the Qt Designer create two required inputs. Use QgsMapLayerComboBox for layer selection, QgsFileWidget for output dir name.

Use objectName attribute to name the objects: layers, output_dir and submit.

Consider using grid layout and horizontal/vertical spacers.

The Code¶

The run() method of the SaveViews class has to do following tasks:

  1. Initialize some initial form inputs

  2. Get the user input data

  3. Call the saving function

 1    def run(self):
 2        """Run method that loads and starts the plugin"""
 3
 4
 5        if not self.pluginIsActive:
 6            self.pluginIsActive = True
 7
 8            #print "** STARTING SaveViews"
 9
10            # dockwidget may not exist if:
11            #    first run of plugin
12            #    removed on close (see self.onClosePlugin method)
13            if self.dockwidget == None:
14                # Create the dockwidget (after translation) and keep reference
15                self.dockwidget = SaveViewsDockWidget()
16                self.dockwidget.layers.setFilters(QgsMapLayerProxyModel.VectorLayer)
17                self.dockwidget.output_dir.setStorageMode(QgsFileWidget.GetDirectory)
18                self.dockwidget.submit.clicked.connect(self.save_views)
19
20            # connect to provide cleanup on closing of dockwidget
21            self.dockwidget.closingPlugin.connect(self.onClosePlugin)
22
23            # show the dockwidget
24            # TODO: fix to allow choice of dock location
25            self.iface.addDockWidget(Qt.LeftDockWidgetArea, self.dockwidget)
26            self.dockwidget.show()

Note, that we had to import QGIS Python modules

1from PyQt5.QtGui import QColor, QPixmap
2from qgis.utils import iface
3from qgis.core import QgsMapLayerProxyModel
4from qgis.gui import QgsFileWidget

Tip

For testing purpose we can hardcode output path

self.dockwidget.output_dir.setFilePath("/tmp/output")

Image save function¶

Finally, we approach to the heart of the plugin: the save_views() method.

 1    def save_views(self):
 2         """
 3         save graphical output for every row in attribute table
 4         """
 5
 6         layer = self.dockwidget.layers.currentLayer()
 7
 8         for feature in layer.getFeatures():
 9             layer.selectByIds([feature.id()])
10             self.iface.mapCanvas().setSelectionColor(QColor("transparent"))
11             box = layer.boundingBoxOfSelected()
12             self.iface.mapCanvas().setExtent(box)
13             pixmap = QPixmap(self.iface.mapCanvas().mapSettings().outputSize().width(),
14                              self.iface.mapCanvas().mapSettings().outputSize().height()
15             )
16             mapfile = os.path.join(
17                     self.dockwidget.output_dir.filePath(),
18                     '{0}_{1:03d}.png'.format(layer.name(), feature.id())
19             )
20             self.iface.mapCanvas().saveAsImage(mapfile, pixmap)
21             layer.removeSelection()
22
23         # save also full extend of vector layer                            
24         canvas = self.iface.mapCanvas()
25         canvas.setExtent(layer.extent())
26         pixmap = QPixmap(self.iface.mapCanvas().mapSettings().outputSize().width(),
27                          self.iface.mapCanvas().mapSettings().outputSize().height()
28         )
29         mapfile = os.path.join(
30                    self.dockwidget.output_dir.filePath(),
31                    '{}_full.png'.format(layer.name())
32        )
33         self.iface.mapCanvas().saveAsImage(mapfile, pixmap)

Task

Make sure that the plugin will not fail, if no target directory is selected.

Hint

In case that you want to change plugin icon, modify the icon.png file as you wish.
