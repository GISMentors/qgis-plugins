Setup QGIS¶

Two helper plugins need to be installed:

  1. Go to Plugins ‣ Manage and Install Plugins… and enable Show also experimental plugins in Settings tab

  2. Search for Plugin Builder 3 and install the plugin

  3. Search for Plugin Reloader and install the plugin
